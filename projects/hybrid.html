Image Filtering and Enhancements

The goal of this project was to explore some image filtering techniques as well as frequency representations.

Hybrid Images

Hybrid images are static images that change in interpretation as a function of the viewing distance. The basic idea is that high frequency tends to dominate perception when it is available, but, at a distance, only the low frequency (smooth) part of the signal can be seen. By blending the high frequency portion of one image with the low-frequency portion of another, you get a hybrid image that leads to different interpretations at different distances.

Filtering the Images

The original images were converted to grayscale for processing. The first image (image of the cat) was filtered using a high pass filter, which was created using a gaussian kernel. The gaussian filtered image was then subtracted from the original image to get laplacian filtered image below. Additionally, we can confirm from the log fequency representation of the image that only the high frequecies remain and the low frequencies (origin of the graph) are filtered out.

The second image (image of the dog) was filtered using a low pass filter, which was created using a gaussian kernel. We can confirm from the log fequency representation of the image that only the low frequecies (origin of the graph) remain and the high frequencies are filtered out.

Result

The result of adding the two images is shown below. When the photo is enlarged, we see the picture of the cat, however, on a smaller scale, the picture of the dog dominates. This same effect would be acheived by physically moving closer/away from the image.

Color Enhancement

We start with an image with 'bad' colors, that is the colors looks faded and are not very sharp. The image colors are improved by first converting the image to the HSV color space. The saturation value is then normalized to the range between 0-1. The square root of the saturation was set as the new saturation value. The Hue and Value channels were left unchanged. This resulted in sharper colors seen on the image to the right.
